## Getting Started

### Prerequisites

- Node.js (v16 or newer)
- npm or yarn package manager

### Installation

1. Clone the repository:

   ```bash
   git clone [repository-url]
   cd anime-schedule
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features in Detail

### Weekly Schedule

The main feature displays anime broadcasts organized by days of the week, showing:

- Anime title and synopsis
- Broadcast time in JST
- Episode information
- Current airing status

### Seasonal Browsing

Explore anime by different seasons and years:

- Winter, Spring, Summer, Fall seasons
- Year-based filtering
- Popular and trending anime highlights

### Search & Discovery

- Real-time search functionality
- Filter by genre, status, and type
- Advanced search options

### Customization

- **Themes**: Multiple visual themes to match user preferences
- **Language**: Multi-language support with translation
- **Layout**: Flexible viewing options

## API Integration

### Jikan API

This application uses the Jikan API to fetch anime data:

- Real-time anime information
- Seasonal anime data
- Search functionality
- No API key required

### LibreTranslate API

For translation features:

- Multiple language support
- Real-time translation
- Fallback to original text if translation fails
